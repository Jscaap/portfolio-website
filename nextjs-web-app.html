About Next.js Web App

This Next.js Dashboard Web App is a project and task management tool built to help users organise their work in a clear and structured way. It acts as a central hub where projects, tasks, milestones, and checklist items can be viewed and maintained in one place. By combining a clean layout with logical navigation and live data from an external API, the app turns complex project data into something easy to understand and work with, supporting users as they plan, track progress, and stay on top of their workload.

My Contributions

My main contributions to the Next.js Dashboard Web App include designing the overall layout and building the key pages for Projects, Tasks, Checklist Items, and Milestones. I implemented the client-side logic to fetch data from the API, display it in a structured way, and handle loading and error states so users always know what is happening. I also set up routing between pages, created reusable components, and focused on responsive styling, ensuring the dashboard works smoothly on different screen sizes while staying visually consistent and easy to use.

Next.js Web App Website Overview

The Home page introduces the dashboard with a friendly welcome message and clear cards that guide users to manage Projects, Tasks, Checklist Items, and Milestones. The Projects page lists all available projects with key details so users can quickly see what they are working on. From there, the Tasks page allows users to view and organise tasks linked to each project, while the Checklist Items page breaks work down into smaller actionable steps. The Milestones page highlights major goals and deadlines, giving an overview of progress. Together, these sections create a structured workflow that supports planning, tracking, and reviewing project work within a single Next.js web application.

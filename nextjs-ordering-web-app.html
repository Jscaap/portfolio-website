About Stock Ordering Web App

The Stock Ordering App is a web-based system designed to simplify and streamline stock ordering for businesses. It allows staff to place orders efficiently while giving managers full visibility and control over approvals. The application focuses on clarity, speed, and usability, helping businesses manage suppliers, products, and orders in one central platform.

My Contributions

I designed and developed the Stock Ordering App from end to end. My work includes building the user interface with a responsive layout, implementing role-based access for staff and boss users, and integrating Supabase for authentication, database management, and security. I also developed core features such as order creation, approval workflows, supplier management, and order history, ensuring the system works smoothly across different user roles.

Stock Ordering Web App Overview

The app begins with a secure login system that directs users to role-specific dashboards. Staff users can create new orders, select suppliers, and submit requests, while boss users can review, approve, or manage orders. The interface is clean and structured, making it easy to navigate through orders, suppliers, and items. The system is designed to be scalable and practical, providing a reliable solution for real-world stock management needs.
